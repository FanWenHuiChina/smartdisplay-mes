## 🐛 常见问题

### 问题1：编译失败
**症状：** maven编译报错
**解决：**
```bash
mvn clean compile
```

### 问题2：数据库连接失败
**症状：** 应用启动报Connection refused
**解决：**
```bash
# 检查Docker是否启动
docker ps

# 重启数据库
docker-compose restart
```

### 问题3：端口被占用
**症状：** Port 8080 is already in use
**解决：**
```bash
# 查找占用端口的进程
netstat -ano | findstr 8080

# 修改application.yml中的端口
server.port: 8081
```
